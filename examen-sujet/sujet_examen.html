Exercice C : Utilisation de Float (4 pts)

Observez le code HTML et écrivez le CSS pour obtenir la mise en page demandée.

Code HTML :

<div class="container">
    <div class="box-gauche">Contenu Gauche</div>
    <div class="box-droite">Contenu Droite</div>
    <div class="texte">
        Lorem ipsum dolor sit amet, consectetur adipiscing elit.
        Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
    </div>
</div>

Résultat attendu :

Contenu Gauche
(float: left, width: 30%)
Contenu Droite
(float: right, width: 30%)
Le texte s'affiche en dessous et occupe toute la largeur (clear: both)

CSS à écrire :

  1. .box-gauche : width 30%, float left, background #f0f0f0, padding 10px
  2. .box-droite : width 30%, float right, background #f0f0f0, padding 10px
  3. .texte : clear both, padding 10px
  4. .container : border 1px solid #333, padding 15px
